void tt_sort ( SparseTensorCOO *const  tt,
IndexType const  mode,
IndexType *  dim_perm 
)

Sort a tensor using a permutation of its modes. Sorting uses dim_perm to order modes by decreasing priority. If dim_perm = {1, 0, 2} then nonzeros will be ordered by ind[1], with ties broken by ind[0], and finally deferring to ind[2].

Parameters
ttThe tensor to sort.
modeThe primary for sorting.
dim_permAn permutation array that defines sorting priority. If NULL, a default ordering of {0, 1, ..., m} is used.
